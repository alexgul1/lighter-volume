# Rate limits

Subscription rate limits

Each IP address is restricted to a maximum of100active websocket connections. Additionally, asingle wallet addresscan be authenticated by up to5websocket connections, regardless of the originating IP address. Connections exceeding these limits will be automatically disconnected.
