# V2

Nado V2 REST API: Gateway and Archive (indexer) Endpoints

Nado V2 API offers REST-based endpoints focused on two key functionalities:

- Gateway Queries: Access real-time market data, including trading pairs and book liquidity.

Gateway Queries: Access real-time market data, including trading pairs and book liquidity.

- Archive (indexer) Queries: Query historical market data, including 24-hour statistics for all products and recent trades.
