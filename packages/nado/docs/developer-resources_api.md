# 📡API

Nado Websocket and REST API

## Overview

Nado's API is divided into the following categories:

- Awebsocket/RESTAPI (gateway) that supports writes (executes) and polling (queries).

Awebsocket/RESTAPI (gateway) that supports writes (executes) and polling (queries).

- AsubscriptionsAPI that allows to subscribe to live data feeds.

AsubscriptionsAPI that allows to subscribe to live data feeds.

- AnindexerAPI (archive) that allows you to query historical data.

AnindexerAPI (archive) that allows you to query historical data.

- AtriggerAPI that allows to execute orders only under specified price conditions.

AtriggerAPI that allows to execute orders only under specified price conditions.
